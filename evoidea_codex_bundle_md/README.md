# evoidea — меметический итератор идей (локально, Rust)

`evoidea` — CLI-инструмент, который по пользовательскому промпту:
1) генерирует популяцию идей,
2) оценивает их по рубрике (мультикритерий),
3) делает отбор (элита + разнообразие),
4) скрещивает и мутирует,
5) улучшает топ,
6) повторяет N итераций до критерия остановки,
7) в конце отдаёт **1 лучшую идею** и пишет артефакты в `runs/<run_id>/`.

Рядом лежит Codex skill: `.codex/skills/evo-ideator`.

## Быстрый старт (mock — детерминированно)
```bash
cargo run -- run --mode mock --prompt "Придумай проект для AI-агентов с быстрым выходом на $10k/m"
```

Результат:
- `runs/<run_id>/config.json`
- `runs/<run_id>/state.json`
- `runs/<run_id>/history.ndjson`
- `runs/<run_id>/final.json`

## Реальный режим (через codex exec)
```bash
cargo run -- run --mode codex --search --prompt "Придумай проект с быстрым выходом на доход 10k/мес"
```
